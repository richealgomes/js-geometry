what is event bubble???

Event bubbling is a method of event propagation in the DOM.
when an event is in an element inside another element,
and both elements have registered a handle to that event.

what is call back function and how can we use call back function???

when you pass another function to a function as a parameter
it is called a callback function. call back function is
only called and executed once a specific event has occurred. As previously mentioned,
callback functions are used in a variety of programming languages.
